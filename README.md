# Менеджер для установки и настройки Hysteria2

## 🔹 Возможности

Hysteria2 Setup Manager делает следующее:

- Устанавливает Hysteria2 на ваш сервер
- Настраивает базовую конфигурацию
- Настраивает fallback-страницу
- Автоматически получает сертификаты для домена

Также, скрипт позволяет:

- Обновить версию Hysteria2 до последней
- Запустить, остановить или перезапустить сервис
- Добавлять, удалять и просматривать пользователей
- Генерировать ссылку для подключения
- Изменить домен
- Изменить E-mail

## 🔹 Необходимые настройки

Перед запуском менеджера убедитесь, что:

- Настроен и защищен доступ к серверу по SSH
- Установлен и настроен firewall (открыты порты 80, 443 и 8443)
- Зарегистрирован и делегирован домен, указывающий на ваш сервер

> [!IMPORTANT]
> Для корректной работы менеджера требуется установленная библиотека **jq**

## 🔹 Запуск

Подключитесь по **SSH** к серверу и выполните команду:

```sh
bash <(wget -O - https://raw.githubusercontent.com/neketabrain/hysteria2-setup-manager/main/hysteria2-setup-manager.sh)
```

или

```sh
wget -O /tmp/hysteria2-setup-manager.sh https://raw.githubusercontent.com/neketabrain/hysteria2-setup-manager/main/hysteria2-setup-manager.sh && bash /tmp/hysteria2-setup-manager.sh
```
